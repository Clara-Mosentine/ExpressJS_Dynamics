What's the difference between div and span?

Div commands "divide" two bodies of text, analogous to pressing the enter key on a text document.
Span commands DON'T "press enter", and can be used in the same line as other elements (hence why they are sometimes called "inline" commands).

The Basics of CSS

What is the basic syntax for setting properties on an element (font, color, etc.)?

Each property on an element can be identified with its name (for instance, color). A colon marks where the property name ends and where the new setting for the property begins. After this, the new property value is given, and the statement is ended with a semicolon (for instance, color:blue;)

How do you use selectors (class, tag, and id)?

You can make classes, tags, and ids by creating the new name of the selector, and then use curly brackets to enclose all of the specifications within. Then, whenever you use the name of that selector within two arrows, the following text will be run under the "rules" of that selector.

What are ways to alter the box model (borders, margin, padding)?

Borders, margins, and padding all help to make your text easier to read. In the case of padding, a text formatting class can specify an indent at the start of any new paragraph. By indicating "padding: X%", you can change X to make the indent as pronounced or subtle as you'd like.
